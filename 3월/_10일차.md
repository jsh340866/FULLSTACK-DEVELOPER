## 오늘 배운 것
    1. 글자 크기 단위 (Font Size)
	- px: 고정된 크기입니다. (기본값 16px)
	- em: 부모 요소의 글자 크기를 기준으로 한 배수입니다.
	- rem: **최상위 요소(root)**의 글자 크기를 기준으로 합니다. (가장 권장되는 단위)
	- %: 부모 요소 크기를 기준으로 한 백분율입니다.
	- vw: 뷰포트(브라우저 화면) 가로폭 기준입니다.
	- clamp(min, val, max): 값의 범위를 지정해, 화면 크기가 변해도 너무 작거나 커지지 않게 제한할 수 있습니다.
	
    2. 여백 관리 (Margin & Padding)
	- margin (바깥 여백): 요소와 요소 사이의 간격을 벌릴 때 사용합니다.
		1개 값: 상하좌우 동일
		2개 값: 위·아래 / 왼쪽·오른쪽
		3개 값: 위 / 왼쪽·오른쪽 / 아래
		4개 값: 위 / 오른쪽 / 아래 / 왼쪽 (시계 방향)
	- padding (안쪽 여백): 요소 내부의 콘텐츠와 테두리 사이의 간격을 조절합니다. 버튼 등을 만들 때 고정 크	
	  기(width)를 지정하기보다 padding을 사용하는 것이 훨씬 유연하고 예쁘게 잡힙니다.

	3. 정렬 공식 (Center Alignment)
	- margin: 0 auto;: 블록 요소를 가로 중앙에 배치할 때 가장 기본적인 방법입니다.
	- display: flex;: 부모 요소에 설정하여 자식 요소를 justify-content와 align-items로 자유자재로 정렬합니다. 	  
	  가장 현대적이고 강력한 방법.
	- position: absolute;: 부모에 relative를 주고, 자식의 상하좌우를 0으로 설정한 뒤 margin: auto;를 주면 
	  정중앙에 배치됩니다.
	  
	4. 테두리 (Border & Radius)
	- border 활용: 투명한 테두리를 사용하여 기하학적 도형(삼각형 등)을 구현할 수 있습니다.
	- border-radius: 모서리를 둥글게 만듭니다. 50% 값을 주면 정사각형을 완벽한 동그라미로 만들 수 있어요.

### FontSize
    <!-- px : 고정 크기 기본값 : 16px -->
    <div style="font-size: 48px;">
        <div style="font-size: 32px;">HELLOWORLD</div>
    </div>
	- 글자 크기의 기본 단위 px


    <!-- em : 상대크기(부모태그의 font-size를 기준으로 한 배수) -->
    <div style="font-size: 24px;">
        <div style="font-size: 1em;">HELLOWORLD</div>
        <div style="font-size: 0.5em;">HELLOWORLD</div>
    </div>
	- 부모의 font-size를 1배수로 계산합니다. em
	

    <!-- rem : 상대크기(기본글자크기 : root 16px기준) -->
    <div style="font-size: 24px;">
        <div style="font-size: 1rem;">HELLOWORLD</div>
        <div style="font-size: 0.5rem;">HELLOWORLD</div>
    </div>
	- :root 또는 <html>의 font-size 기준입니다. rem


    <!-- % : 상대크기(상위태그기준) -->
    <div style="font-size: 24px;">
        <div style="font-size: 200%;">HELLOWORLD</div>
        <div style="font-size: 75%;">HELLOWORLD</div>
    </div>
	- 부모 크기에 대한 백분율로 계산됩니다. %


    <!-- vw : 상대크기(뷰포트 기준) -->
    <div style="font-size: 24px;">
        <div style="font-size: 1vw;">HELLOWORLD</div>
        <div style="font-size: 1.5vw;">HELLOWORLD</div>
    </div>
	- 화면 가로폭 전체를 100으로 나눈 값입니다. vw


    <!-- clamp(최솟값,비율,최댓값) -->
    <div style="width:clamp(200px,50vw,1280px);height: 200px;border: 1px solid;margin: 0 auto;">
        <div style="font-size: clamp(0.8rem,5vw,4rem); text-align: center; line-height: 200px;">
            HELLOWORLD
        </div>
	- vw를 사용할 때 너무 작아지거나 커지지 않게 최솟값과 최댓값을 고정할 수 있다.

### Margin
        /* 1. 가운데맞추기 - margin : 0 auto */
        .d2 {
            margin: 50px auto;
        }

        /* 2. 가운데맞추기 - display:flex (LAYOUT)상위태그에서 사용 */
        .parent1 {
            height: 250px;
            width: 80vw;
            margin: 0 auto;

            display: flex;
            flex-direction: column;
            justify-content: center;
            align-items: center;
            gap: 20px;
        }

        .son {
            background-color: royalblue;
            width: 150px;
            height: 150px;
            border: 1px solid;
        }

        /* 3. 가운데맞추기 : position */
        .parent2 {
            height: 250px;
            width: 80vw;
            margin: 0 auto;
            /*  */
            position: relative;
        }
        /* 가운데 배치 공식 */
        .parent2>.son{
            position: absolute;
            left : 0;
            right : 0;
            top: 0;
            bottom: 0;
            margin: auto;
        }
