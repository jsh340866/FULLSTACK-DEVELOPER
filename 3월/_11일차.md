## 오늘 배운 것
```
  1. relative 과 absolute
     - relative (상대적): "내가 원래 있어야 할 자리"를 기준으로 움직여요.
      이동해도 원래 자리는 비워두지 않고 그대로 차지하고 있어서 다른 요소들이 밀리지 않아요.
     - absolute (절대적): "가장 가까운 부모(static이 아닌)"를 기준으로 움직여요.
      만약 부모 중에 기준이 없으면 화면 전체(Viewport)를 기준으로 삼고, 공중에 붕 뜨기 때문에 원래 자리를 차지하지 않아요.

  2. GNB(Global Navigation Bar) 연습.  
    1) 구조 설계 (HTML)  
     - 3단계 계층: main-menu (1단계) → sub-menu (2단계) → last-menu (3단계)로 이어지는 구조.

    2) 스타일링 전략 (CSS)  
     - 포지셔닝: position: relative (부모)와 position: absolute (자식)를 조합하여    
       하위 메뉴가 상위 메뉴 옆에 자연스럽게 붙도록 설계.(left right로 조절)

     - 상태 제어: display: none으로 숨겨두었다가, :hover 가상 클래스를 이용해 마우스를 올리면
       display: block으로 나타나게 하는 전형적인 드롭다운 구현 방식

    3) 코드의 특징  
     - Flexbox 사용: main-menu를 flex-direction: column으로 배치해서 세로형 메뉴 구조를 만드셨네요.
       (만약 가로형 메뉴를 원하신다면 이 부분을 row로 바꾸거나 수정이 필요할 수 있어요!)

     - hover를 활용해 마우스를 올리면 display: none을 display: block으로 바꾸어 메뉴가 나타나게 설계.

    4) display none은 transition 함께 쓸수없다.
     - display: none과 display: block 사이에는 transition이 적용되지 않습니다.
       (브라우저가 중간 단계를 계산할 수 없기 때문이에요!)

     - 아래 코드를 사용해 사용할 수 있음.
     - opacity (0 ↔ 1): 투명도를 조절해서 스르륵 나타나게 합니다.
       visibility (hidden ↔ visible): 메뉴가 안 보일 때 마우스 클릭이 되지 않도록 확실히 막아줍니다.
       transition: opacity 속성에 0.3s 정도의 시간을 주면 아주 부드러운 메뉴가 완성됩니다.
  3. Fixed, Sticky
     - position : fixed;
     - 화면(브라우저 창)에 껌딱지처럼 딱 붙어서, 스크롤을 아무리 내려도 그 자리에 그대로 있는 속성
     - position: sticky;
     - 평소에는 스크롤을 따라 같이 올라가다가, 설정한 위치(예: top: 0)에 닿는 순간 그 자리에 고정.


  4. Zindex
     - z-index: 5;
     - 기본적으로 나중에 쓴 코드(아래쪽 코드)가 위로 올라오지만, z-index 숫자를 높여주면 강제로 맨 앞으로 끌어올릴 수 있어요.

  5. 방향설정, 배치설정
     - display: flex :	Flex 컨테이너 선언	"지금부터 정렬을 시작하겠다!"

     - flex-direction	: 정렬 방향 결정	가로로 세울지, 세로로 쌓을지 정하기

     - justify-content : 메인축(기본 가로) 정렬	자식들을 왼쪽, 오른쪽, 혹은 가운데 어디에 둘까?

     - align-items : 교차축(기본 세로) 정렬	자식들의 키를 맞추거나 세로 중앙에 두기

     - flex-wrap : 줄바꿈 여부	자리가 부족하면 다음 줄로 보낼까 말까?

  6. 추가 설정
     - flex-grow: 1	 : 형제간 공간 나눠먹기 비율	남는 공간을 5명이서 1:1:1:1:1로 나눠 가짐

     - flex-shrink: 0	 : 줄어듦 거부	화면이 작아져도 내 크기(200px)를 지킴

     - flex-basis	: 기본 크기	200px이라는 최소한의 자기 영역 확보

     - align-self :	나 혼자 정렬 방식 변경	부모의 정렬 무시하고 위/아래로 개별 이동 



```
