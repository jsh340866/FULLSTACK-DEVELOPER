## 오늘 배운 것
    VSCode DAY 06 CSS 참조
    
    1. FLEX 핵심
      - flex 계열의 태그를 사용하기 위해서는 display:flex;를 먼저 사용해야한다.

    2. Grid
      - display: grid: 웹 페이지를 바둑판처럼 격자로 나누어 관리할 수 있게 해줘요.
      - grid-template-columns: 화면 가로를 몇 칸으로 나눌지 정해요.
              ex) grid-template-columns: repeat(3, 1fr)
              ex> grid-template-columns: 1fr 1fr 1fr
      - grid-template-areas: 구역에 이름을 붙여서 위치를 직관적으로 배치해요.
      - auto-fit + minmax: 화면 크기가 변해도 칸들이 알아서 예쁘게 정리되게 해요.

    3. 브라우저 ICON 링크 가져올때 팁

      - <link rel="stylesheet" href="https://fonts.googleapis.com/css2?...100..700,0..1,-50..200" />
