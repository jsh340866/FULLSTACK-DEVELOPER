## 오늘 배운 것
    VSCode DAY 06 CSS 참조
    
    1. FLEX 핵심
      - flex 계열의 태그를 사용하기 위해서는 display:flex;를 먼저 사용해야한다.

    2. Grid
      - display: grid: 웹 페이지를 바둑판처럼 격자로 나누어 관리할 수 있게 해줘요.
      - grid-template-columns: 화면 가로를 몇 칸으로 나눌지 정해요.
              ex) grid-template-columns: repeat(3, 1fr)
              ex> grid-template-columns: 1fr 1fr 1fr
      - grid-template-areas: 구역에 이름을 붙여서 위치를 직관적으로 배치해요.
      - auto-fit + minmax: 화면 크기가 변해도 칸들이 알아서 예쁘게 정리되게 해요.

    3. 브라우저 ICON 링크 가져올때 팁
      - <link rel="stylesheet" href="https://fonts.googleapis.com/css2?...100..700,0..1,-50..200&icon_names=menu" />
      - <link rel="stylesheet" href="https://fonts.googleapis.com/css2?...100..700,0..1,-50..200" />
      - &icon_names=menu 이부분만 지워서 관련 아이콘을 전부 가져올수 있다!

    4. Transition 지연효과
      - transition: 2s;
      - 어떤 속성값이 바뀔 때, 순식간에 바뀌지 않도록 시간 차를 주는 설정
      - duration(지속 시간), delay(지연 시간), timing-function(속도 곡선).

    5.  Transform 형태변화
      - transform: translateY(850px);
      - 요소의 형태를 비틀거나, 돌리거나, 옮기는 등 기하학적인 변화를 줘요
      - rotate(45deg): 회전하기
      - scale(1.5): 크기 조절하기
      - translate(10px, 20px): 위치 이동하기
      - skew(10deg): 비틀기

    6. Animation 
      - animation-name: 사용할 대본의 이름
      - animation-duration: 애니메이션이 한 번 실행되는 데 걸리는 시간 (예: 3s)
      - animation-iteration-count: 몇 번 반복할지 (예: infinite는 무한 반복!)
      - animation-direction: 앞뒤로 왔다 갔다 할지 정하기 (alternate)
      - Animation은 @keyframes로 작성한 시간별 대본에 따라 요소의 스타일을 복잡하고 역동적으로 변화시키는 기능

    7. Keyframe
      - @keyframes moving1 {
        from {margin-left: 100%}
        to {margin-left: 0%;}
        }
      - 시작과 끝만 정할 때: from (시작)과 to (끝)를 사용해요.
      - 중간 단계를 정할 때: 0%, 50%, 100%처럼 퍼센트로 아주 세밀하게 쪼개서 변화를 줄 수 있어요.
