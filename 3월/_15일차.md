1. 변화 감지하기 (Event Listener)
catRestorant.addEventListener('change', (e) => { ... })
풀이: 사용자가 카테고리 선택창(드롭다운)에서 다른 항목을 클릭하는 순간, 자바스크립트가 "앗! 값이 바뀌었네?" 하고 알아차리는 부분이에요.

e: 이벤트 정보를 담고 있는 바구니예요.

2. 선택한 값 가져오기
console.log(e.target.value);
const target = e.target.value;
풀이: 사용자가 '한식'을 골랐다면 target 변수에는 "한식"이라는 글자가 담겨요.

3. 데이터 걸러내기 (Filter)
const selectItems = 중구맛집_data.filter((item) => {
    return (target == 'all') ? item : target.includes(item.FD_CS);
})
풀이: 전체 맛집 리스트 중에서 내가 고른 카테고리에 맞는 것만 쏙쏙 골라내요.
target == 'all': 전체를 골랐다면 모든 데이터를 통과!
target.includes(item.FD_CS): 내가 고른 카테고리 이름이 데이터의 카테고리(FD_CS)와 일치하는지 확인해요.
결과: selectItems에는 필터링된 맛집 객체들의 배열이 담깁니다.

4. 기존 목록 지우기 (Cleaning)
while (selectRestorantEl.firstChild) {
    selectRestorantEl.removeChild(selectRestorantEl.firstChild);
}
풀이: 상호명 선택창에 이미 다른 식당들이 들어있을 수 있겠죠? 새 음식을 담기 전에 그릇을 깨끗이 비우는 과정이에요.
첫 번째 자식 노드(firstChild)가 없을 때까지 계속해서 지워버립니다.

5. 새 목록 채우기 (Rendering)
selectItems.forEach((item) => {
    const optionEl = document.createElement('option');
    optionEl.innerHTML = item.BZ_NM;
    selectRestorantEl.appendChild(optionEl);
})
풀이: 필터링해서 골라낸 맛집들(selectItems)을 하나씩 꺼내요.
createElement('option'): 새로운 <option> 태그를 하나 만들어요.
innerHTML = item.BZ_NM: 그 태그 안에 맛집 이름(BZ_NM)을 써넣어요.
appendChild(optionEl): 완성된 옵션을 상호명 선택창(selectRestorantEl)에 쏙 집어넣습니다.
