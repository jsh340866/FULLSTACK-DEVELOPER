     *** 함수 기본! ***
     1. 전통적인 방식 (function 키워드)
    문법: function 함수명(인자) { ... }
    특징: 호이스팅(Hoisting)이 적용되어 함수 선언 전에도 호출이 가능하고, this 바인딩 방식이 화살표 함수와 달라요.


    2. 현대적인 방식 (화살표 함수 const)
    문법: const 함수명 = (인자) => { ... }
    특징: 1.  간결함: 코드가 훨씬 짧아져요. (특히 리턴만 있을 때 return 생략 가능!)
          2.  안전성: const를 사용하기 때문에 함수 이름을 실수로 덮어쓰는 일을 방지할 수 있어요.
          3.  익명 함수: 변수에 할당하는 방식이라 구조가 깔끔해요.


    형태              예시 코드                              특징 
    인자 O, 리턴 O    (n1, n2) => n1 + n2                    값을 받아서 계산한 뒤 결과를 돌려줘요. (가장 많이 써요!)
    인자 O, 리턴 X    (n1, n2) => { console.log(n1+n2) }     값을 받아서 출력하거나 실행만 하고 끝나요.
    인자 X, 리턴 O    () => { return n1 + n2 }               외부 값은 안 받지만, 내부(prompt 등)에서 만든 값을 돌려줘요.
    인자 X, 리턴 X    () => { console.log(n1+n2) }           스스로 입력받고 스스로 출력하고 끝나는 독립적인 실행형이에요.
