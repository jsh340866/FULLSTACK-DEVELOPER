    *** ScrollMagic ***

  if (spyedEl.classList.contains('first')) {
    const ballEl = spyedEl.querySelector('.ball');
    //스크롤 매직 객체
    new ScrollMagic
      .Scene({                                            //감시 위치 옵션
        triggerElement: spyedEl,                         //감시할 요소
        triggerHook: 0.5,                                //스크롤 위치값(0-1, 0:최상단, .5:중앙, 1:최하단)
      })
        .setClassToggle(ballEl, 'move1')                      //애니메이션 효과 적용
        .addTo(new ScrollMagic.Controller());               //제어부(스크롤 상황 감시)
  }
  1. 주요 구성 요소
     - Controller (제어부): 스크롤 상태를 감시하는 전체적인 관리자예요.
                            페이지당 하나만 생성해서 모든 Scene을 여기에 추가(addTo)하여 사용해요.
     - Scene (장면): "어디서, 어떤 일이 일어날지"를 정의하는 개별 단위예요.
                    특정 요소가 화면에 보일 때 클래스를 넣었다 뺐다 하거나 애니메이션을 실행하죠.
     
  2. Scene의 주요 옵션
    옵션                  설명
    triggerElement        애니메이션이 시작될 기준이 되는 요소를 지정해요.
    triggerHook           뷰포트(화면) 내의 기준 지점을 정해요. (0: 최상단, 0.5: 중앙, 1: 최하단)
    duration              효과가 지속될 거리(px)를 설정해요. 설정하지 않으면 일회성 트리거가 돼요.
    offset                트리거 지점에서 얼마나 더 스크롤한 뒤 시작할지 정해요.

  4. 주요 메서드
     setClassToggle(요소, '클래스명'): 지정한 요소에 특정 클래스를 스크롤 위치에 따라 넣었다(.add) 뺐다(.remove) 해줘요. 
     addTo(Controller): 생성한 Scene을 관리자(Controller)에 등록해서 실제로 작동하게 만들어요.
     setPin(요소): 스크롤하는 동안 특정 요소를 화면에 고정시켜요.

    *** GSAP *** 
    gsap.to('.box1', { duration: 3, x: 200, y: 200, opacity: 0.3, ease: 'steps(10)', delay: 2, });
    - 주요 파라미터(옵션) 설명
    속성명       설명                                    비고
    duration    애니메이션이 지속되는 시간(초)            기본값은 0.5초예요.
    delay       애니메이션이 시작되기 전 대기 시간        코드의 box1처럼 2초 뒤 시작 가능!
    ease        가속도(타이밍 함수) 조절elastic          (통통 튀기), steps(10)(끊기기) 등
    repeat      반복 횟수 설정                           -1을 넣으면 무한 반복돼요!
    yoyo        왔던 길을 되돌아가는 효과                 repeat와 함께 쓰면 찰떡궁합이에요.

    *** QRCODE ***
    [ qrcode.js 사이트 코드 참고 ]
      <script>
        // QR 코드 생성
    
        var qr = qrcode(0, 'L'); // 오류 정정 레벨(Low) //qrcode(version, errorCorrectionLevel)
        qr.addData('https://www.naver.com'); // QR 코드에 포함될 데이터
        qr.make();
    
        // QR 코드를 HTML 요소로 렌더링
        var qrDiv = document.getElementById('qrcode');
        qrDiv.innerHTML = qr.createSvgTag({ scalable: true });
        // QR 코드를 클릭하면 URL로 이동
        qrDiv.addEventListener('click', function () {
          window.location.href = 'https://www.naver.com';
        });
      </script>
