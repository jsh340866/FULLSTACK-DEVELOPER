### Java 기본 구조 (`HelloWorld.java`)

#### 1. 핵심 개념
* **Main 메서드**: 프로그램 실행 시 가장 먼저 호출되는 '입구' 역할을 하는 절차지향 문법 적용 영역이에요.
* **메서드 구분**: 미리 제공되는 '라이브러리 메서드'와 개발자가 직접 만드는 '사용자 정의 메서드'로 나뉘어요.

#### 2. 꼭 외울 패턴
* **클래스 선언**: `public class 클래스명 { ... } - 객체지향문법 적용영역`
* **실행부**: `public static void main(String[] args) { ... } - 절차지향문법 적용영역`

### 표준 출력 함수 (`C01SystemOut.java`)

#### 1. 핵심 개념
* **서식 문자(Format)**: `%d`, `%f`, `%s` 등을 사용하여 정수, 실수, 문자열을 원하는 형식으로 출력할 수 있어요.
* **이스케이프 문자**: `\n`(줄바꿈), `\t`(탭 이동) 등을 사용해 출력 형태를 제어해요.

#### 2. 꼭 외울 패턴
* **줄바꿈 출력**: `System.out.println("내용");`
* **서식 출력**: `System.out.printf("%d.%s : %d\n", 1, "국어", 100);`

---


### 데이터 표현 및 진법 (`C02진수.java`)


#### 1. 핵심 개념
* **진법 표기**: 자바에서는 `0b`(2진수), `0`(8진수), `0x`(16진수) 접두사를 사용해 숫자를 표현해요.
* **데이터 단위**: 1bit는 2가지 상태를 표현하며, 8bit($2^8$)가 모여 1byte(256가지 상태)가 돼요.

#### 2. 꼭 외울 패턴
* **진수 변환 출력**:
    * `%d`: 10진수로 출력
    * `%o`: 8진수로 출력
    * `%x`: 16진수로 출력

---

### 음수와 보수 개념 (`C03음수.java`)

#### 1. 핵심 개념
* **2의 보수**: 컴퓨터는 뺄셈을 처리하기 위해 '2의 보수 + 가산 처리' 방식을 사용해요.
* **계산 과정**: 어떤 수의 '1의 보수'(비트 반전)를 구한 뒤, 1을 더하면 해당 수의 음수 표현(2의 보수)이 돼요.

#### 2. 꼭 외울 패턴
* **음수 변환 예시**:
    * `5` (00000101) → 1의 보수 (11111010) → 2의 보수 (11111011, `-5`)
    * `96` (01100000)  → 1의 보수 (10011111) → 2의 보수 (10100000, `-96`)
