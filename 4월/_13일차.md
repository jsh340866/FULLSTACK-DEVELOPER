### Java 자료형 및 상수 (`C06상수.java`)

#### 1. 핵심 개념
* [cite_start]**기본 자료형**: 정수는 `int`, 실수는 `double`이 기본이며, `long`은 `L`, `float`은 `f` 접미사를 붙여야 해요. [cite: 1, 3]
* **상수의 종류**: 이름 없는 '리터럴 상수'와 `final` 예약어를 써서 이름을 붙인 '심볼릭 상수'로 나뉘어요.

#### 2. 꼭 외울 패턴
* **심볼릭 상수 선언**: `final double PI = 3.14;` (값 변경 불가)
* [cite_start]**리터럴 접미사 사용**: `float var = 3.14f;`, `long var = 50000000000L;` [cite: 3]

---

### 형변환 원리 (`C01TypeChange.java` ~ `C03TypeChange.java`)

#### 1. 핵심 개념
* **자동 형변환**: 범위가 좁은 곳에서 넓은 곳으로 대입될 때 컴파일러가 자동으로 처리해 줘요.
* **강제 형변환**: 범위가 넓은 곳에서 좁은 곳으로 넣을 때 `(자료형)`을 명시하며, 데이터 손실이 발생할 수 있어요.

#### 2. 꼭 외울 패턴
* **강제 형변환(Casting)**: `int intValue = (int)doubleValue;`
* **자동 형변환 순서**: `byte` → `short`/`char` → `int` → `long` → `float` → `double`

### 확인 문제 정리 (`C07정리문제`)

#### 1. 핵심 개념
* [cite_start]**유효 범위**: `byte`는 -128 ~ 127까지만 담을 수 있어 200 같은 숫자는 대입할 수 없어요. [cite: 2]
* [cite_start]**문자열 특수문자**: 문자열 안에서 큰따옴표를 출력하려면 `\"` 이스케이프 시퀀스를 사용해야 해요. [cite: 5]

#### 2. 틀리기 쉬운 코드
* [cite_start]`char var = 'AB';` (X): `char`는 한 문자만 가능해요. [cite: 2]
* [cite_start]`boolean var = 0;` (X): 자바의 `boolean`은 `0/1`이 아닌 `true/false`만 가능해요. [cite: 5]
* [cite_start]`float var = 3.14;` (X): 기본이 `double`이므로 `f`를 안 붙이면 에러가 나요. [cite: 3]
