## 프로그래밍 언어 - Java 형변환 심화 및 Scanner 입출력

## 오늘자 주요 태그

* **형변환**: 자동 형변환(Promotion) / 강제 형변환(Casting) / 연산식 형변환
* **입력 처리**: Scanner 클래스 / 표준 입력 스트림(System.in)
* **주의사항**: `nextLine()` 버퍼 비우기 / 데이터 손실 / 정수 연산 결과(`int`)

## 오늘 배운 것

* **중요**: `C01TypeChange.java` ~ `C07TypeChange_String.java`, `C01Scanner.java` ~ `C03Scanner.java`
* **알아두기**: `Scanner` 객체는 사용 후 `sc.close()`로 닫아주는 것이 메모리 관리상 좋아요.
* **핵심**: `nextInt()`나 `nextDouble()` 뒤에 `nextLine()`을 쓸 때는 반드시 버퍼에 남은 엔터(`\n`)를 제거해야 해요.

### Java 형변환(Type Change) 핵심 정리

#### 1. 자동 및 강제 형변환
* **자동 형변환**: 작은 타입에서 큰 타입으로 대입될 때 컴파일러가 자동으로 처리하며, 데이터 손실이 최소화되는 방향으로 결정돼요.
* **강제 형변환**: 큰 타입을 작은 타입에 저장할 때 사용하며, 캐스팅 연산자 `()`를 명시해야 해요. 이 과정에서 상위 비트가 잘려나가는 데이터 손실이 발생할 수 있어요.

#### 2. 연산식에서의 타입 변환
* **정수 연산**: `byte`, `short`, `char` 타입의 변수끼리 산술 연산을 하면 결과는 자동으로 **`int`** 타입이 돼요.
* **실수 연산**: 정수와 실수가 연산되면 더 큰 타입인 실수 타입으로 자동 변환되어 계산돼요.
* **문자열 결합**: 문자열과 `+` 연산을 하면 다른 피연산자도 문자열로 변환되어 결합돼요. 연산 순서에 따라 결과가 달라지니 주의가 필요해요.

### Scanner를 이용한 데이터 입력 (`CH03`)

#### 1. Scanner 기초
* **개념**: `System.in`(표준 입력 스트림)을 연결하여 사용자가 키보드로 입력한 데이터를 읽어오는 객체예요.
* **객체 생성**: `Scanner sc = new Scanner(System.in);`
* **주요 메서드**:
    * `nextInt()`: 정수 값을 읽어옴 (공백이나 엔터로 구분)
    * `nextDouble()`: 실수 값을 읽어옴
    * `next()`: 공백 문자 전까지의 문자열을 읽어옴
    * `nextLine()`: 엔터(`\n`) 전까지의 한 줄 전체 문자열을 읽어옴

#### 2. [중요] nextLine() 사용 시 주의점
* `nextInt()` 등은 숫자만 읽고 입력 버퍼에 엔터(`\n`)를 남겨둬요.
* 뒤이어 `nextLine()`을 호출하면 버퍼에 남은 엔터를 입력으로 인식해 빈 문자열을 반환하고 종료돼요.
* **해결법**: 중간에 `sc.nextLine();`을 한 번 추가로 호출하여 버퍼에 남은 엔터를 비워줘야 해요.

#### 3. 입력 활용 예시
* 이름은 `nextLine()`, 나이는 `nextInt()`, 그리고 다시 주소를 입력받기 전에는 버퍼 비우기용 `nextLine()`을 사용한 후 실제 주소를 입력받는 패턴을 사용해요.
