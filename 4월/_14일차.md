## 오늘자 주요 태그

* **산술 연산**: 기본 사칙연산 / `%` 연산자의 다양한 활용(배수, 끝자리 구하기)

### Java 연산자(Operator) 핵심 정리

#### 1. 증감 연산자의 마법
* **전치 연산 (`++a`)**: "먼저 증가, 나중 연산". 변수의 값을 1 먼저 증가시킨 후 해당 라인의 나머지 연산을 수행해요.
* **후치 연산 (`a++`)**: "먼저 연산, 나중 증가". 현재 변수 값으로 연산을 모두 마친 뒤, 세미콜론(`;`)을 넘어가기 직전에 값을 1 증가시켜요.
* **주의**: 한 줄에 여러 개의 증감 연산자가 복합적으로 쓰이면 결과 예측이 어려우므로 코드 가독성을 위해 나누어 쓰는 것이 좋아요.



#### 2. `%` (나머지) 연산자의 4대 활용
* **짝홀수 구분**: `N % 2 == 0`이면 짝수, 아니면 홀수로 판별해요.
* **배수 확인**: `N % M == 0`이면 N은 M의 배수임을 의미해요.
* **수 범위 제한**: `N % 10`의 결과는 항상 `0~9` 사이의 값만 가져요. (난수 범위 지정 시 유용)
* **자리수 추출**: `num % 10`은 오른쪽 끝 한 자리를, `num / 10`은 오른쪽 끝 자리를 제외한 나머지를 얻을 때 사용해요.

#### 3. 논리 연산의 효율성 (단락 회로)
* **`&&` (논리곱)**: 앞의 조건이 `false`이면 뒤의 조건은 쳐다보지도 않고 전체를 `false`로 처리해요.
* **`||` (논리합)**: 앞의 조건이 `true`이면 뒤의 조건이 무엇이든 전체를 `true`로 처리하고 종료해요.
* **실전 팁**: `(n1 < 100) && (n2++ > 0)` 같은 식에서 `n1 < 100`이 거짓이면 `n2++`는 실행되지 않아 `n2` 값이 그대로 유지될 수 있어요.

#### 4. 복합 대입 및 삼항 연산자
* **복합 대입**: `a += 10`은 `a = a + 10`과 동일하며, 코드를 간결하게 만들어 줘요.
* **삼항 연산자**: `(조건식) ? 참 : 거짓;` 형태의 연산자로, 단순한 `if-else`문을 대체하여 변수에 값을 바로 대입할 때 매우 유용해요.
