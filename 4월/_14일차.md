### 1. [Ch04] Java 연산자(Operator) 핵심 정리

#### 증감 연산자의 마법
* **전치 연산 (`++a`)**: 먼저 1을 증가시킨 후 나머지 연산을 수행해요.
* **후치 연산 (`a++`)**: 현재 값으로 연산을 모두 마친 뒤, 다음 문장으로 넘어가기 직전에 1을 증가시켜요.

#### `%` (나머지) 연산자의 활용
* 짝수/홀수 판별(`n % 2 == 0`), 배수 확인, 수 범위 제한(0~N-1), 그리고 자리수 추출 등에 사용돼요.

#### 논리 연산과 단락 회로 (Short-Circuit)
* `&&`는 앞이 `false`면 뒤를 실행 안 하고, `||`는 앞이 `true`면 뒤를 실행하지 않아요. 이 때문에 뒤쪽의 증감 식(`++b` 등)이 무시될 수 있으니 주의가 필요해요.

#### 비트 연산자 및 시프트
* `&`, `|`, `^`, `~`(and,or,xor,not)등 비트 단위 연산과 `<<`, `>>` 시프트 연산을 통해 메모리 효율적인 계산이 가능해요.

### 2. [Ch05] Java 조건문(Condition Statement) 핵심 정리

#### if-else 및 중첩 if
* **if-else if**: 여러 조건 중 만족하는 단 하나의 블록만 실행해요.
* **중첩 로직**: 국어, 영어, 수학 등 여러 과목의 과락 여부와 평균 점수를 동시에 판단할 때 효과적이에요.



[Image of Java if-else and else-if flowchart]


#### 선택문 (switch-case)
* 변수의 값(정수, 문자, 문자열)에 따라 실행 위치를 결정해요.
* **주의**: `case` 끝에 `break`가 없으면 다음 `case`까지 연달아 실행되는 'Fall-through' 현상이 발생해요.

---

### 3. [Ch06] Java 반복문(Loop Statement) 핵심 정리

#### while 반복문의 구조
* **반복의 3요소**: 
    1. **초기값** (변수 선언)
    2. **조건식** (반복 범위)
    3. **증감식** (탈출을 위한 변화)
* 조건식이 `true`인 동안 실행문을 반복하며, 증감식이 누락되면 무한 루프에 빠지게 돼요.



#### 반복문과 선택문의 조합
* `while(true)` 내부에 `switch-case`를 넣으면 사용자가 '종료'를 선택하기 전까지 계속 동작하는 메뉴 방식의 프로그램을 만들 수 있어요.
* 특정 조건에서 `System.exit(-1)`을 호출하면 프로그램 전체를 즉시 종료할 수 있습니다.
