## 프로그래밍 언어 - Java 기초 (반복문 while & 별 찍기 고급)

## 오늘자 주요 태그

* **반복 제어**: 탈출 변수 / 조건식 / 연산식(i++)
* **다중 반복**: 중첩 while / 행과 열(i, j) / 추가 매개변수(k)
* **패턴 분석**: 공백 수식 `j < (h-1)-i` / 별 개수 수식 `k <= i*2`
* **논리 구조**: 바깥 루프(행) -> 안쪽 루프 1(공백) -> 안쪽 루프 2(별)

## 오늘 배운 것

* **중요**: `C01WHILE.java`, `C02WHILE.java`
* **알아두기**: 공백과 별이 섞인 패턴은 `j`로 공백을 먼저 밀어내고, `k`로 별을 찍는 **'순차적 안쪽 루프'** 구조를 이해하는 것이 핵심이에요!
* **수식 활용**: 행 번호(`i`)가 바뀜에 따라 변하는 공백과 별의 개수를 `i`를 포함한 수식으로 만드는 연습을 했습니다.

### 반복문의 정의와 중첩 구조 (`C01WHILE.java`, `C02WHILE.java`)

#### 1. 핵심 개념
* **다중 안쪽 루프**: 하나의 바깥쪽 `while`문(행) 안에 여러 개의 독립적인 `while`문(열)을 배치하여, 한 줄 내에서 '공백'과 '별'을 순서대로 출력할 수 있습니다.
* **k 매개변수의 역할**: 주로 `j`가 공백을 담당할 때, 이어서 별을 담당하여 출력 횟수를 제어하는 보조 제어 변수로 활용됩니다.

#### 2. 꼭 외울 패턴 (k 매개변수 활용 피라미드)
* **피라미드 (높이 h 기준)**:
  ```java
  int i = 0;
  while (i < h) {
      // 1) 공백 출력 (j 활용)
      int j = 0;
      while (j < (h - 1) - i) {
          System.out.print(" ");
          j++;
      }
      
      // 2) 별 출력 (k 활용)
      int k = 0;
      while (k <= i * 2) { // 홀수 개수로 증가 (1, 3, 5, 7...)
          System.out.print("*");
          k++;
      }
      
      System.out.println();
      i++;
  }

* **옆으로 누운 피라미드**:
  ```java
  Scanner sc = new Scanner(System.in);
  int h = sc.nextInt(); // 전체 높이 입력
  int i = 1; // 행 번호
  int k = 1; // 별 개수 제어 변수
  
  while (i < (h + 1)) {
      int j = 1;
      // j/2 < k 조건에 따라 별 출력 (약 2k개 내외)
      while (j / 2 < k) {
          System.out.printf("*");
          j++;
      }
      System.out.println();
      
      // 중간 지점 도달 여부에 따라 k값 증감 결정
      if (i < (h + 1) / 2) {
          k++; // 상단부: 별 개수 증가
      } else {
          k--; // 하단부: 별 개수 감소
      }
      i++;
  }
sc.close();
