## 프로그래밍 언어 - Java 기초 (반복문 while & 제어 알고리즘)

## 오늘자 주요 태그

* **반복 제어**: 탈출 변수(i) / 조건식 / 연산식(i++)
* **데이터 처리**: 누적 합산(sum) / SWAP / 입력값 유효성 검증
* **중첩 루프**: 다중 while / 구구단 / 별 찍기 패턴
* **동적 제어**: if-else를 이용한 매개변수(k) 증감 / 대칭형 패턴 구현

## 오늘 배운 것

* **중요**: `C01WHILE.java`, `C02WHILE.java` 및 조건부 별 찍기 로직
* **알아두기**: 반복문은 단순히 횟수를 채우는 것이 아니라, 조건에 따라 제어 변수(`k`)를 조절하여 복잡한 흐름을 만드는 것이 핵심이에요!
* **핵심 원리**: 안쪽 루프의 조건식(`j/2 < k`)과 분기점(`i < (h+1)/2`)의 조화를 통해 알고리즘의 효율성을 높일 수 있습니다.

---

### 1. 반복문의 기초와 누적 연산 (`C01WHILE.java`)

#### 핵심 개념
* **while 기본**: 조건식이 `true`인 동안 실행되며, 무한 루프 방지를 위해 반드시 탈출 연산식이 필요합니다.
* **데이터 누적**: `sum += i`와 같은 패턴으로 범위 내 합계를 구하며, `if`문을 섞어 홀수/짝수별 합계도 구할 수 있습니다.
* **SWAP**: `n > m`일 때 두 값을 교환하여 반복문의 시작과 끝을 안정적으로 고정합니다.

---

### 2. 중첩 반복문과 패턴 출력 (`C02WHILE.java`)

#### 핵심 개념
* **다중 루프**: 바깥쪽 `while`은 '행'을, 안쪽 `while`은 '열'을 제어합니다.
* **패턴 수식**: 
    * **직사각형**: 고정 횟수 반복
    * **삼각형**: `j <= i` (행 번호에 의존)
    * **피라미드**: `j < (h-1)-i` (공백)와 `k <= i*2` (별)의 조합

---

### 3. 조건부 매개변수 제어 (Advanced Pattern)

#### 핵심 개념
* **동적 증감**: `if-else`를 사용하여 반복문 내부가 아닌 '외부'에서 제어 변수 `k`의 값을 상황에 따라 변화시킵니다.
* **분기점 설정**: `(h+1)/2`를 기준으로 증감 방향을 전환하여 상하 대칭 구조를 형성합니다.

#### 핵심 패턴 코드
* **피라미드 (높이 h 기준)**:
  ```java
  int i = 0;
  while (i < h) {
      // 1) 공백 출력 (j 활용)
      int j = 0;
      while (j < (h - 1) - i) {
          System.out.print(" ");
          j++;
      }
      
      // 2) 별 출력 (k 활용)
      int k = 0;
      while (k <= i * 2) { // 홀수 개수로 증가 (1, 3, 5, 7...)
          System.out.print("*");
          k++;
      }
      
      System.out.println();
      i++;
  }

* **옆으로 누운 피라미드**:
  ```java
  Scanner sc = new Scanner(System.in);
  int h = sc.nextInt(); // 전체 높이 입력
  int i = 1; // 행 번호
  int k = 1; // 별 개수 제어 변수
  
  while (i < (h + 1)) {
      int j = 1;
      // j/2 < k 조건에 따라 별 출력 (약 2k개 내외)
      while (j / 2 < k) {
          System.out.printf("*");
          j++;
      }
      System.out.println();
      
      // 중간 지점 도달 여부에 따라 k값 증감 결정
      if (i < (h + 1) / 2) {
          k++; // 상단부: 별 개수 증가
      } else {
          k--; // 하단부: 별 개수 감소
      }
      i++;
  }
  sc.close();
