### 1. 반복문 심화 및 알고리즘 (`JAVA01/Ch06`)

#### 핵심 개념
* **중첩 while**: 바깥쪽 루프가 한 번 돌 때 안쪽 루프가 전체를 반복하는 구조를 이용하여 구구단이나 행렬 구조를 표현합니다. (`C02WHILE.java`)
* **입력값 검증**: `while(조건)`을 활용해 사용자가 올바른 값을 입력할 때까지 반복 질문하는 방어적 로직을 구현합니다.
* **SWAP 로직**: 두 변수의 값을 바꿀 때 임시 변수(`tmp`)를 활용하여 데이터 유실을 방지합니다. (`C01WHILE.java`)

#### 주요 별 찍기 수식 요약
| 패턴 | 공백(j) 수식 | 별(k) 수식 | 비고 |
| :--- | :--- | :--- | :--- |
| **피라미드** | `j < (h-1)-i` | `k <= i*2` | i는 0부터 시작 |
| **다이아몬드(상)** | `j < (h/2)-i` | `k <= 2*i` | `if(i <= h/2)` |
| **다이아몬드(하)** | `j < i-h/2` | `k <= ((h-1)*2)-i*2` | `else` 구간 |

### 2. 꼭 외울 코드 패턴

#### 다이아몬드 상하 대칭 분기 로직
```java
if (i <= h / 2) {
    // 상단 증가 로직 (공백 감소, 별 증가)
    int j = 0;
    while (j < (h / 2) - i) { System.out.print(" "); j++; }
    int k = 0;
    while (k <= 2 * i) { System.out.print("*"); k++; }
} else {
    // 하단 감소 로직 (공백 증가, 별 감소)
    int j = 0;
    while (j < i - h / 2) { System.out.print(" "); j++; }
    int k = 0;
    while (k <= ((h - 1) * 2) - i * 2) { System.out.print("*"); k++; }
}
```

### 3. 객체 지향 프로그래밍의 기초 (`JAVA02/Ch01`)

#### 핵심 개념
* **클래스(Class)**: 동일한 종류의 객체에 필요한 메모리 공간을 정의한 자료형입니다. (`C01Person.java`)
* **객체(Object)**: 클래스를 바탕으로 `new` 연산자를 통해 Heap 영역에 생성된 실체입니다. (`C02PersonMain.java`)
* **메서드(Method)**: 객체의 기능을 정의하며, 입력(Parameter)과 출력(Return)의 유무에 따라 4가지 형태로 나뉩니다. (`C04MethodMain.java`)
* **변수의 유효 범위**:
    * **멤버 변수**: 클래스 내부에서 객체의 속성을 저장.
    * **지역 변수**: 메서드나 특정 블록(`{ }`) 내에서만 유효하며 Stack에 저장. (`C05LocalVarMain.java`)

### 4. 자바 클래스의 기본 구조 (`JAVA02/Ch01`)

#### 클래스와 메서드 설계
* **C01Person.java**: 이름(`name`), 나이(`age`) 등 사람의 속성을 필드로 정의합니다.
* **C04MethodMain.java**: 계산기(`Cal`) 클래스를 통해 매개변수와 리턴값의 유무에 따른 4가지 메서드 형태를 학습합니다.
* **C05LocalVarMain.java**: 멤버 변수와 지역 변수의 차이점을 다룹니다.

```java
class Simple {
    int num = 10; // 멤버 변수 (Heap 영역)

    void Func1() {
        int num = 20; // 지역 변수 (Stack 영역)
        System.out.println("지역변수: " + num);
        System.out.println("멤버변수: " + this.num);
    }
}
```
