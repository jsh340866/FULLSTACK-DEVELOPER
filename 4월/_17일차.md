### 1. 메서드 오버로딩과 가변 인자

#### 핵심 개념

* **오버로딩(Overloading)**: 같은 이름의 메서드를 매개변수 형태만 다르게 여러 개 만드는 방식입니다.

예시:

```java
sum()
sum(int a, int b)
sum(int a, int b, int c)
```

* **가변 인자**: 전달되는 인수 개수가 일정하지 않을 때 사용합니다.

```java
sum(int ...arg)
```

* 배열처럼 받아 반복문으로 누적 합계를 구할 수 있습니다.

#### 핵심 포인트

* 이름은 같아도 **매개변수 개수·자료형이 다르면 다른 메서드**로 인식됩니다.
* 사용자 편의성이 높아집니다.

---

### 2. 생성자와 객체 초기화

#### 핵심 개념

* 생성자는 객체 생성 시 자동 호출되는 특별한 메서드입니다.

```java
Student s = new Student();
```

* `new Student()`에서 `Student()`가 생성자입니다.

#### 특징

* 클래스 이름과 동일해야 합니다.
* 반환형이 없습니다.
* 객체의 초기값 설정에 사용됩니다.

#### 예제 흐름

```java
C08Simple() {
    this.n1 = 11;
}
```

### 7. 문자열 비교 (`String`)

#### 핵심 개념

```java
str1 == str2
str1.equals(str2)
```

* `==` : 주소 비교
* `.equals()` : 내용 비교

예시:

```java
String a = "java";
String b = "java";
```

```java
new String("java")
```
