## 1. 문자열 기본 메서드 (`C05StringMain.java`)

### 핵심 개념

* 문자열 길이 확인 → `length()`
* 특정 위치 문자 확인 → `charAt(index)`
* 특정 문자 위치 찾기 → `indexOf()`, `lastIndexOf()`
* 포함 여부 확인 → `contains()`
* 문자열 자르기 → `substring()`

### 예제 코드

```java
String str1 = "Java Powerful";

System.out.println(str1.length());
System.out.println(str1.charAt(2));
System.out.println(str1.indexOf('a'));
System.out.println(str1.lastIndexOf('a'));
System.out.println(str1.contains("va"));
System.out.println(str1.substring(2,6));
```

### 예상 출력

```java
13
v
1
3
true
va P
```

## 2. 문자열 분리 (`split()`)

### 핵심 개념

* 특정 구분자를 기준으로 문자열을 잘라 배열로 저장합니다.
* CSV 데이터, 사용자 입력값 처리 등에 자주 사용됩니다.

### 예제 코드

```java
String str = "등산,탁구,축구,골프,독서,영화감상";
String[] result = str.split(",");

for(String hobby : result)
    System.out.println(hobby);
```

### 예상 출력

```java
등산
탁구
축구
골프
독서
영화감상
```

## 3. String 과 StringBuilder 차이

### 핵심 개념

| 구분 | 특징 |
|------|------|
| String | 값 변경 시 새 객체 생성 |
| StringBuilder | 기존 객체 내부 수정 |
| StringBuffer | Builder와 같지만 동기화 지원 |

### 문제 상황

```java
String str = "";
str += "A";
str += "B";
str += "C";
```

* 문자열이 바뀔 때마다 새 메모리가 생성됩니다.

### 개선 코드

```java
StringBuilder sb = new StringBuilder();

sb.append("A");
sb.append("B");
sb.append("C");

System.out.println(sb);
```

### 예상 출력

```java
ABC
```

## 4. 클래스 실습 - Profile 객체 (`C06Ex.java`)

### 핵심 개념

* 사용자 정보를 하나의 객체로 관리
* 멤버 변수는 `private` 처리 (캡슐화)
* 생성자를 통해 값 저장
* `toString()`으로 객체 출력 형식 지정

### 멤버 변수 예시

```java
private String name;
private String addr;
private String job;
private String major;
```

## 5. 생성자 오버로딩

### 핵심 개념

하나의 클래스에 여러 생성자를 만들 수 있습니다.

### 예제 코드

```java
Profile hong = new Profile("홍길동,대구,프로그래머,컴퓨터공학");
```

### 처리 방식

```java
String[] arr = str.split(",");
name = arr[0];
addr = arr[1];
job = arr[2];
major = arr[3];
```

## 6. 문자열 검색 메서드 구현

### isContain()

입력 문자열이 이름, 주소, 직업, 전공 중 하나라도 포함되면 `true`

### 예제

```java
hong.isContain("길동");
hong.isContain("컴퓨터");
```

### 예상 출력

```java
true
true
```

---

## 7. 프로필 비교 메서드 구현

### isEquals()

입력값:

```java
"홍길동,대구,프로그래머,컴퓨터공학"
```

객체 내부 값과 전부 일치하면 `true`

### 예제

```java
hong.isEquals("홍길동,대구,프로그래머,컴퓨터공학");
hong.isEquals("홍길동,울산,프로그래머,컴퓨터공학");
```

### 예상 출력

```java
true
false
```
