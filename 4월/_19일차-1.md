* **배열 기본(Ch03)**: `new` 연산자를 이용한 메모리 할당 / 인덱스 참조 / `length` 속성
* **메모리와 복사**: 얕은 복사(Shallow Copy) vs 깊은 복사(Deep Copy) 실습
* **다차원 배열**: 행마다 길이가 다른 가변 배열(Ragged Array) 처리

* **중요**: 배열은 생성 시 크기가 고정되는 정적 자료구조이며, `arr.length`를 통해 요소의 개수를 동적으로 파악할 수 있어요.
* **알아두기**: 2차원 배열에서 `arr.length`는 행의 개수, `arr[i].length`는 해당 행의 열 개수를 의미해요.
* **핵심**: 객체 배열은 참조 변수들의 묶음일 뿐이므로, `list[i] = new C04Person();`과 같이 각 인덱스에 실제 객체를 생성해 넣어줘야 해요.

### 1. [Ch03] Java 배열(Array) 및 복사 매커니즘

#### 1차원 배열과 데이터 추출 (C01ArrayMain.java)
* `Scanner`로 입력받은 값을 배열에 저장하고 반복문을 통해 분석하는 실습을 진행했어요.
```java
// 최댓값 구하기 예시
int max = arr1[0];
for (int i = 0; i < arr1.length; i++) {
    if (max < arr1[i]) {
        max = arr1[i]; // 현재 저장된 max보다 큰 값이 나오면 교체
    }
}
```

#### 얕은 복사 vs 깊은 복사 (C02ArrayMain.java)
* 얕은 복사: arr2 = arr1;은 주소만 공유하기 때문에 arr1[0] = 100;을 실행하면 arr2[0]도 함께 바뀌어 버려요.
* 깊은 복사: 별도의 메모리 공간을 확보하여 값을 직접 복사해요.

```java
// C02ArrayMain.java 실습 코드
// 1. for문을 이용한 직접 값 복사
int arr3[] = new int[3];
for (int i = 0; i < arr1.length; i++)
    arr3[i] = arr1[i]; 

// 2. Arrays API 사용
int arr4[] = Arrays.copyOf(arr1, arr1.length);
```

### 2. [Ch03] 다차원 배열과 가변 배열
* 가변 배열의 이해 (C03ArrayMain.java)
* 자바의 2차원 배열은 행마다 열의 개수가 다를 수 있음을 코드로 확인했어요.
  
```java
int arr[][] = {
    {10, 20, 30},              // arr[0].length : 3
    {40, 50, 60, 65, 67},      // arr[1].length : 5
    {70, 80, 90, 96, 11, 56}   // arr[2].length : 6
};
```

### 3. [Ch03] 객체 배열 및 메인 파라미터
#### 객체 배열 관리 (C04ClassArrayMain.java)
* 객체 배열 생성 주의사항:
  1. C04Person list[] = new C04Person[3]; (참조 변수 공간 3개 생성)

* 객체를 생성하지 않고 멤버 변수(name, age)에 접근하면 NullPointerException이 발생해요.

#### 메인 메서드 매개변수 (C05MainParam.java)
* public static void main(String[] args)의 args 배열을 통해 실행 시 외부 인자를 받을 수 있어요.

```Java
System.out.println("length : " + args.length); // 전달된 인자의 개수 확인
for(String param : args) {
    System.out.println(param); // 인자 내용 순차 출력
}
```
