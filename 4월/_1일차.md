## 오늘 배운 것

    MySQL 폴더 참조

```
    *** MySQL cmd 명렁어 ***
    - 대소문자 구분을 하지않음
    - 명령어가 끝날때 세미콜론(';') 달아줄 것
    
    1. 데이터베이스 생성 및 선택
    CREATE DATABASE test1db;  -- 'test1db'라는 이름의 데이터베이스를 생성해요.
    USE test1db;             -- "지금부터 이 데이터베이스를 사용할게!"라고 선택해요.


    2-A. 데이터베이스 생성 
    CREATE TABLE tbl_user (
      userId VARCHAR(45) PRIMARY KEY,  -- 아이디를 만들면서 바로 기본키로 지정!
      userName VARCHAR(45) NULL,
      password VARCHAR(255) NULL
    );

    2-B 데이터베이스 생성 
    CREATE TABLE tbl_user (
      userId VARCHAR(45) NOT NULL,
      userName VARCHAR(45) NULL,
      password VARCHAR(255) NULL,
      PRIMARY KEY (userId)             -- 모든 컬럼을 적고 난 뒤 기본키를 지정!
    );


    3. 구조 확인 및 수정 (CMD 활용)
    SHOW DATABASES;: 현재 내 컴퓨터에 만들어진 모든 데이터베이스 목록을 확인해요.
    DESC tbl_user;: 생성한 테이블의 설계도를 한눈에 확인해요. (컬럼명, 타입 등)
    
    ALTER TABLE ... ADD COLUMN ... AFTER: 테이블 열(colunm) 추가
    ALTER TABLE ... DROP COLUMN ... AFTER: 테이블 열(colunm) 삭제
        예시)
        - ALTER TABLE tbl_user ADD COLUMN phone VARCHAR(45) NULL AFTER userName;
          이 명령어는 userName 뒤에 phone 칸을 쏙 집어넣어 줘요.
