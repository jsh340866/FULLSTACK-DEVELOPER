# 26-04-29

## 프로그래밍 언어 - Java(`JAVA02/Ch07`) 추상클래스(Abstract), 인터페이스(Interface), final 심화 정리

## 오늘자 주요 태그

* **추상클래스**: `abstract class`, 추상 메서드, 구현 강제
* **인터페이스**: 다형성, 공통 규격, 다중 구현
* **업캐스팅**: 부모 타입 참조변수로 자식 객체 제어
* **다형성**: 같은 타입으로 서로 다른 객체 동작
* **의존성 분리**: 부품 교체 구조
* **final**: 상수, 오버라이딩 금지, 상속 금지

---

## 오늘 배운 것

오늘은 Java 객체지향의 핵심 개념인 **추상클래스**, **인터페이스**, **final 키워드**를 학습했습니다.  
직접 객체를 만드는 클래스뿐 아니라, **설계 전용 클래스**, **공통 기능 규약**, **변경을 막는 구조**까지 다루며 실무형 객체지향 구조를 이해하는 단계입니다.

---

## 1. 추상클래스(Abstract Class)

### 설명

추상클래스는 **완성되지 않은 클래스**입니다.  
공통 구조는 만들되, 일부 기능은 자식 클래스가 반드시 구현하도록 강제합니다.

```java
abstract class Parent2 {
    abstract void func();
}
```
