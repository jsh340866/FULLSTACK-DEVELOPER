### 📚 정렬 알고리즘 상세 설명 노트

#### 1. 버블 정렬 (Bubble Sort)
마치 물속의 거품이 올라오듯, 인접한 두 원소를 비교하며 정렬하는 방식입니다.
* **작동 방식:** 앞에서부터 두 개씩 짝지어 비교하여 큰 값을 뒤로 보냅니다.
* **회전 특징:** 1회전(Pass)이 끝나면 **가장 큰 값**이 맨 뒤로 고정됩니다.
* **시간 복잡도:** $O(n^2)$ (데이터가 많으면 매우 느려짐)

#### 2. 선택 정렬 (Selection Sort)
해당 자리에 넣을 최솟값을 '선택'해서 가져오는 방식입니다.
* **작동 방식:** 전체 데이터 중 최솟값을 찾아 첫 번째 자리에 놓고, 그다음 최솟값을 찾아 두 번째 자리에 놓는 과정을 반복합니다.
* **회전 특징:** 1회전이 끝나면 **가장 작은 값**이 맨 앞에 고정됩니다.
* **시간 복잡도:** $O(n^2)$

#### 3. 삽입 정렬 (Insertion Sort)
자신의 위치를 찾아 '삽입'하는 방식입니다.
* **작동 방식:** 두 번째 자료부터 시작하여, 앞쪽의 정렬된 자료들과 비교하며 자신의 위치를 찾아 들어갑니다.
* **장점:** 데이터가 이미 거의 정렬된 상태라면 매우 효율적입니다.
* **시간 복잡도:** $O(n^2)$ (최선일 때 $O(n)$)

#### 4. 합병 정렬 (Merge Sort)
분할 정복(Divide and Conquer) 알고리즘의 대표 주자입니다.
* **작동 방식:** 데이터를 더 이상 쪼갤 수 없을 때까지 반으로 나눈 뒤, 다시 합치면서(Merge) 순서대로 정렬합니다.
* **장점:** 어떤 데이터가 들어와도 성능이 일정하며, 대용량 데이터 처리에 좋습니다.
* **시간 복잡도:** $O(n \log n)$

### 📑 정보처리 필기 핵심 요약 노트

### 1. 정렬 알고리즘 (Sorting)

#### 🎀 주요 4대 정렬 비교
| 정렬 방식 | 핵심 원리 | 시간 복잡도 |
| :--- | :--- | :--- |
| **버블 정렬** | 인접한 두 값을 비교하며 큰 값을 뒤로 보냄 | $O(n^2)$ |
| **선택 정렬** | 최솟값을 찾아 맨 앞의 값과 교환 | $O(n^2)$ |
| **삽입 정렬** | 정렬된 부분에 자신의 위치를 찾아 삽입 | $O(n^2)$ |
| **합병 정렬** | 반으로 나누고 다시 합치며 정렬 (분할 정복) | $O(n \log n)$ |

### 📑 데이터베이스 일반 집합 연산 요약

#### 1. 카티션 프로덕트 (Cartesian Product, $R \times S$)
두 릴레이션에 있는 모든 튜플을 수평으로 결합하는 연산입니다.

* **차수 (Degree, 열):** $R$의 차수 + $S$의 차수 (**더하기**)
* **카디널리티 (Cardinality, 행):** $R$의 카디널리티 × $S$의 카디널리티 (**곱하기**)

#### 2. 합집합 (Union, $R \cup S$)
두 릴레이션의 합을 구하며, 중복된 튜플은 한 번만 기술합니다.
* **조건:** 합병 가능 조건(동일 차수, 동일 도메인)을 만족해야 합니다.
* **특징:** 결과의 차수는 변하지 않습니다.

#### 3. 차집합 (Difference, $R - S$)
첫 번째 릴레이션에는 있고 두 번째 릴레이션에는 없는 튜플을 구합니다.
* **조건:** 합병 가능 조건을 만족해야 합니다.
* **특징:** 순서에 따라 결과가 달라지는 비가환적 연산입니다.

#### 4. 교집합 (Intersection, $R \cap S$)
두 릴레이션에 공통으로 존재하는 튜플들만 추출합니다.
* **특징:** 합집합, 차집합과 마찬가지로 합병 가능해야 연산이 가능합니다.

---

#### 💡 요약표 (시험 암기용)

| 연산자 | 기호 | Degree(열) 변화 | Cardinality(행) 변화 |
| :--- | :---: | :--- | :--- |
| **합집합** | $\cup$ | 변화 없음 | 합산 후 중복 제거 |
| **교집합** | $\cap$ | 변화 없음 | 공통 부분만 남음 |
| **차집합** | $-$ | 변화 없음 | 앞 집합 - 교집합 |
| **카티션** | $\times$ | **더하기 (+)** | **곱하기 ($\times$)** |
