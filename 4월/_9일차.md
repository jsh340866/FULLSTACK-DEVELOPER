### 호스트 주소의 영역
00000000 : 호스트 주소의 영역
하나의 ip에 네트워크를 여러개 연결해야할 경우
마지막 00000000 의 값을 앞에서부터 1로 수정한다.

#### 1. **1/0000000** 2개 이하의 네트워크가 필요한 경우
* 0과 1, 2개의 네트워크가 생성
* 호스트는 0-126 ,(0000000-1111111)
* 127은 브로드캐스트 (호스트 마지막 번호는 브로드캐스트가 차지한다)

#### 2. **11/000000** 4개 이하의 네트워크가 필요한 경우
* 0과 1, 2*2개의 네트워크 생성
* 호스트는 0-62 ,(000000-111111)
* 63은 브로드캐스트 (호스트 마지막 번호는 브로드캐스트가 차지한다)

### 🐧 정보처리기사 대비 리눅스 & 네트워크 핵심 명령어 정리

#### 1. 파일 및 디렉토리 관리
* **ls**: 현재 디렉토리 파일 목록 보기 (`ls -l`은 상세 보기)
* **cd**: 디렉토리 이동
* **pwd**: 현재 위치한 경로 출력
* **mkdir**: **디렉토리 생성**
* **rmdir**: 비어있는 디렉토리 삭제
* **cp**: 파일/디렉토리 복사
* **mv**: 파일/디렉토리 이동 및 이름 변경
* **rm**: 파일/디렉토리 삭제 (`rm -r`은 디렉토리 강제 삭제)
* **cat**: 파일 내용 출력 (리눅스)
* **type**: **윈도우(CMD)에서 파일 내용 출력** / 리눅스에서는 명령어의 유형 확인

#### 2. 권한 및 소유권 (★실기 단골)
* **chmod**: **파일 보호 권한 설정** (예: `chmod 755 파일명`)
* **chown**: **소유자 및 그룹 변경** (예: `chown 사용자명:그룹명 파일명`)

#### 3. 시스템 및 프로세스 관리
* **ps**: 현재 실행 중인 프로세스 목록 출력
* **top**: 실시간 시스템 리소스 사용량 확인
* **kill**: **프로세스 강제 종료** (PID 사용)
* **uname**: 시스템 정보 및 커널 버전 확인
* **df / du**: 디스크 남은 공간 / 디렉토리별 사용 용량 확인

#### 4. 환경 변수 및 출력
* **echo**: 문자열이나 변수값 출력 (`echo $변수명`)
* **export**: **환경 변수 설정** 및 하위 프로세스 전달
* **env**: 모든 환경 변수 목록 보기

#### 5. 네트워크 및 보안 (★중요도 높음)
* **nmap**: **네트워크 포트 스캐닝** 및 보안 취약점 진단 도구
* **ftp**: 원격 서버 간 **파일 전송 프로토콜** 및 명령어
* **ping**: 네트워크 연결 상태 확인 (ICMP 프로토콜 사용)
* **netstat**: 네트워크 연결 상태, 라우팅 테이블, 인터페이스 통계 확인

#### 6. 기타 유틸리티
* **grep**: 파일 내 **특정 패턴/문자열 검색**
* **find**: 파일 위치 찾기
* **tar**: 파일 묶기(아카이브) 및 압축 해제
* **man**: 명령어 도움말(매뉴얼) 확인
